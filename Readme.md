# JobPortal1

🎓 Bu proje, bitirme ödevi kapsamında geliştirilmiş bir iş ilanı ve başvuru sistemidir. ASP.NET Core MVC mimarisi kullanılarak tasarlanmış ve Entity Framework Core ile veri yönetimi sağlanmıştır.

## 🔧 Özellikler

- 👤 Aday kayıt ve giriş sistemi
- 💼 İş ilanı oluşturma ve listeleme
- 📝 Adayların ilanlara başvuru yapabilmesi
- 📬 İletişim mesajları gönderme
- 🗃️ Veritabanı yönetimi (EF Core)

## 🧱 Kullanılan Teknolojiler

- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL (veya başka bir veritabanı)
- C#
- Visual Studio / VS Code
